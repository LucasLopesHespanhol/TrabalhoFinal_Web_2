Trabalho interdisciplinar onde aprendemos novas tecnologias e ferramentas, além do trabalho em grupo abordando git e github para ter a disponibilidade do versionamento.

As tecnologias envolvidas foram:

- HTML
- CSS
- JavaScript/React.js
- Node.js
- Express.js
- Mongodb
